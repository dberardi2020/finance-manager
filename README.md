# finance-manager

A personal finance tracker for logging purchases and subscriptions. Built with Vue 2, Vuetify, and Firebase.

## Features
- Track one-time purchases and recurring subscriptions
- Breakdown view by category
- Firebase auth and Firestore backend

## Stack
- **Frontend:** Vue 2, TypeScript, Vuetify
- **Backend:** Firebase (Auth + Firestore)

## Setup
1. Copy `.env.example` to `.env` and fill in your Firebase project credentials
2. Install dependencies and run:
```sh
npm install
npm run serve
```
